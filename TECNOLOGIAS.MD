Listo. Solo tecnología + una línea del porqué, corto y claro:

Django – Framework principal en Python para manejar lógica del ecommerce, órdenes y simulación de pagos.

Django Templates – Renderizado del frontend sin necesidad de React.

PostgreSQL – Base de datos robusta para productos, usuarios, órdenes y pagos.

Django ORM – Modelado y consultas a la base de datos de forma simple y segura.

Tailwind CSS – Estilos modernos y consistentes sin escribir CSS tradicional.

daisyUI – Componentes UI listos (botones, cards, modals) similares a shadcn pero sin React.

HTMX – Interacciones dinámicas (carrito, checkout) sin convertir la app en una SPA.

Alpine.js – Lógica ligera para micro-interacciones en la interfaz.

Django Admin – Panel administrativo para gestionar productos, órdenes y pagos.

Docker – Entorno reproducible para desarrollo y despliegue del proyecto.
